Какие навыки вы получаете

Блок HTML + CSS + личный проект

  1. Понимаете, как работает браузер и веб - в целом
  2. Можете работать с Photoshop, как верстальщик
  3. Умеете писать правильный семантический HTML-код
  4. Умеете добавлять стили CSS веб-странице
  5. Умеете создавать адаптивную и отзывчивую верстку
  6. Знаете CSS-методологии и инструменты автоматизации

Какие навыки вы получаете

Блок JavaScript + 2 проекта ( личный и в команде)

  1. Знаете базовые принципы и подходы программирования
  2. Умеете решать стандартные задачи в ежедневной разработке
  3. Умеете "оживлять" верстку: "карусели", модальные окна
  4. Умеете собирать интерфейс по динамическим данным от Back-end
  5. Можете разбивать код на модули и использовать webpack
  6. Знаете JavaScript для изучения фреймворков

Какие навыки вы получаете

Блок React JS + проект в команде

  1. Понимаете, что такое веб-приложение и как работает
  2. Умеете создавать клиентскую часть SPA-приложения, используя JavaScript фреймворк
  3. Знаете, как создать routing на стороне клиента
  4. Умеете управлять состоянием приложения
  5. Умеете создавать адаптивную и отзывчивую верстку
  6. Умеете тестировать и отлаживать приложения

Какие навыки вы получаете

Блок Back-end + проект в команде

  1. Умеете написать сервер без фреймворка
  2. Знаете, как использовать Back-end фреймворк Express
  3. Можете управлять базой данных, MongoDB
  4. Связываете Back-end и Front-end

Какие навыки вы получаете

Блок Soft Skills

  1. Умеете работать в Agile-командах
  2. Умеешь работать по Scrum со всеми спринтами, митингами, грумингом и демо с ретро
  3. Умеешь работать в Jira, Trello и с тулами прототипирования
  4. Умеешь работать с BackLog, User Stories, MVP и планированием релизов
